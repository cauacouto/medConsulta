MedConsulta API
--
Descrição

A MedConsulta é uma API em construção que permite aos usuários agendar consultas e visualizar as especialidades oferecidas por uma clínica. O objetivo é proporcionar um sistema simples e eficiente para o gerenciamento de consultas médicas.

Funcionalidades
--
Cadastro de pacientes

Listagem de especialidades médicas

Agendamento de consultas

Listagem de consultas agendadas

Cancelamento de consultas

Tecnologias Utilizadas
--
Backend:  Java (Spring Boot)

Banco de Dados: PostgreSQL
